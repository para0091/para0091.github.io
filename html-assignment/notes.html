Body tags
Everything that we see on a webpage is enclosed in the body tags. Therefore, all the content that we want to show in the page should be put inside the body tags.
Heading tags
We can use different heading tags to showcase various headings. There are 6 heading tags, from h1 to h6.
Image
In order to show images on our website, we have to include an img tag. We should also add an src attribute to source the image from a location. Adding an alt attribute is also useful as it will show what the image contains if it is missing or not loading.
Lists
There are three types of lists which can be used in HTML. They are unordered lists (ul), ordered lists (ol) and description lists (dl). All of these should be inside a list tag (li).
Forms
We can add forms to our website to accept input from the user. We do so by using the form tag. We use label, input, select, textarea, checkbox and radio tags to accept different types of input from the user. We also use a button tag to allow the users to submit a form.
